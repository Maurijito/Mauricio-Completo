Propiedad "Position"

La pripiedad "Positionsition" especifica el tipo de método de posicionamiento utilizado para un elemento, es decir que le dice al navegador en que posicion poner el objeto que se este manipulando.

Sus valores son:

  • tatic
  • relative
  • absolute
  • fixed
  • sticky


Valor por defecto:static
Hereditario: no
Animable: no
Version: CSS2v

Propiedad "width"

La propiedad "width" establece el ancho de un elemento. El ancho de un elemento no incluye relleno, bordes o márgenes. Nota: Las propiedades "min-width" y "max-width" anulan la width propiedad.

Sus valores son:

  • Auto (Valor por defecto:El navegador calcula el margen.
  • length: Define el ancho en px, cm,etc.
  • %: Define el ancho en relacion al ancho completo de la pagina.
  • initial:Establece un valor predeterminado.
  • inherit: Hereda esta propiedad de su elemento principal.


Valor por defecto:Auto
Hereditario:No
Animable:Si
Version:CSS1

Propiedad "height"

La propiedad "height" establece la altura de un elemento. La altura de un elemento no incluye relleno, bordes o márgenes.

Sus valores son:

  • Auto: El navegador calcula el margen.
  • Length: Especifica un margen en px, pt, cm, etc
  • %: Especifica un margen en% del ancho del elemento contenedor.
  • Initial
  • Inherit: Especifica que el margen debe heredarse del elemento primario.


Valor por defecto:Auto
Hereditario:No
Animable:Si
Version:CSS1

Propiedad "margin"

Las marginpropiedades CSS se usan para crear espacio alrededor de elementos, fuera de cualquier borde definido.

Sus valores son:

  • Auto: El navegador calcula el margen.
  • Length: Especifica un margen en px, pt, cm, etc
  • %: Especifica un margen en% del ancho del elemento contenedor.
  • Inherit: Especifica que el margen debe heredarse del elemento primario.


Propiedad "padding"

Las propiedades "padding" de CSS se usan para generar espacio alrededor del contenido de un elemento, dentro de cualquier borde definido. Con CSS, tienes control total sobre el relleno. Existen propiedades para configurar el relleno para cada lado de un elemento (arriba, derecha, abajo e izquierda).

Sus valores son:

  • Top
  • Right
  • Bottom
  • Left
